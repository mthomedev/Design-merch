# Design Merch — UI/UX Portfolio

> A UI/UX exercise project built with pure HTML, CSS and JavaScript.

![Design Merch Preview](https://img.shields.io/badge/HTML-CSS-purple?style=for-the-badge&logo=html5&logoColor=white)
![Status](https://img.shields.io/badge/status-complete-success?style=for-the-badge)

## 🖥️ Demo

Open `index.html` in your browser or host it on GitHub Pages.

## 📁 Structure

```
design-merch/
├── index.html   # Semantic and accessible markup
├── style.css    # Design system with CSS variables + responsive layout
├── main.js      # Interactions: mobile menu, form, scroll reveal, i18n
└── README.md
```

## ✨ Features

- **Design system** with CSS custom properties (variables)
- **Typography** with Playfair Display + DM Sans
- **Responsive** — mobile-first with breakpoints at 480px, 768px and 1200px
- **Accessibility** — HTML5 semantics, `aria-*`, `role`, `:focus-visible`
- **Animations** — animated blobs, staggered hero fade-in, scroll reveal
- **Mobile menu** — hamburger with animated transition to ✕
- **Functional form** — validation, visual feedback, auto re-enable
- **Smart nav** — active link per visible section + backdrop blur on scroll
- **Multi-language** — EN / PT / ES / FR switcher with localStorage persistence
- **`prefers-reduced-motion`** — respects system accessibility settings

## 🛠️ Technologies

| Technology   | Usage                                        |
| ------------ | -------------------------------------------- |
| HTML5        | Semantic structure                           |
| CSS3         | Custom properties, Flexbox, Grid, animations |
| JavaScript   | DOM, IntersectionObserver, i18n, events      |
| Google Fonts | Playfair Display, DM Sans                    |

## 🚀 How to run

```bash
# Clone the repository
git clone https://github.com/mthomedev/Design-merch

# Enter the folder
cd design-merch

# Open in your browser
open index.html
# or drag the file into the browser
```

## 📦 Deploy (GitHub Pages)

2. Under **Source**, select `Deploy from a branch`
